## USER
最重要的部分，目前计划

- 静态：`Vue+MD`  
	- iview
- API: `PHP` or `njs`

- 框架

```
      2nd        +--------+
        +-------->  User  |
        |        |        +---------+ 1st
        |        +--+--^--+         |
+-------+--+        |  |        +---v-------------+
| User API |        |  |        | Static FrontEnd |
|          <--------+  +--------+                 |
+----------+                    +-----------------+

```
为什么要这样做？
- Static 可以托管，减缓需要的流量，防止通过拖流量的方法占用服务器宽带，减少成本
- 分开部署，API轻量级随意平行拓展
